Passos para Hospedar um projeto non GitHub Pages

  1. Crie um repositorio no GitHub com o mesmo nome do seu projeti
  2. No terminal, inicialize o repositorio local com git init
  3. Altere o nime do branch principal para main
  4. adicione os arquivos com git add . e faça o commit com git commit -m "first commit".
  5. Conecte ao repositorio remoto com: git remote add origin htttp://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.GIT
  6. Envie os arquivos com git push -u origin main
  7. no GitHub, vá em Settings > PAges
  8. Em "Branch", slecione main e a pasta / (root). Clique em Save
  9. Apos alguns segundos, seu site estará disponivel no link exibido em pages
